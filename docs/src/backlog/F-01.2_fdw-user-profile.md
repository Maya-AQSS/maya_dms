**Descripción:**
Como sistema
Quiero consultar el perfil completo del usuario activo (nombre, email, departamento, grupos académicos) desde la base de datos corporativa vía FDW
Para mostrar información contextual y aplicar filtros de visibilidad sin duplicar datos de usuarios en la base de datos local.

**Criterios de Aceptación (Checklist):**

- [ ] **Escenario 1 (Consulta filtrada por usuario):**
  **Dado que** el sistema necesita el perfil del usuario autenticado,
  **Cuando** ejecuta la consulta a la tabla foránea `users_fdw`,
  **Entonces** la consulta siempre incluye el filtro `WHERE id = :user_id` extraído del JWT; nunca se ejecuta un SELECT sin filtro de usuario.

- [ ] **Escenario 2 (Caché del perfil):**
  **Dado que** el perfil del usuario ha sido consultado,
  **Cuando** el sistema almacena el resultado,
  **Entonces** el perfil queda cacheado en Redis con la clave `user_profile:{user_id}` y un TTL de 15 minutos.

- [ ] **Escenario 3 (Fallback ante indisponibilidad FDW):**
  **Dado que** el FDW no responde en 500 ms,
  **Cuando** el sistema no puede obtener el perfil completo,
  **Entonces** usa los datos mínimos disponibles en el payload del JWT como fallback y la operación continúa con esa información parcial.

- [ ] **Escenario 4 (Prohibición de JOIN sin filtro):**
  **Dado que** se realiza una consulta que involucra la tabla `users_fdw`,
  **Cuando** el código es revisado o se ejecuta la consulta,
  **Entonces** no existe ningún JOIN entre tablas locales y `users_fdw` sin que incluya el filtro del usuario activo como condición.

- [ ] **Requisito Técnico (Rendimiento):**
  Una consulta a `users_fdw` sin caché debe responder en menos de 200 ms en el percentil 95. Con caché Redis activo, la misma consulta debe responder en menos de 5 ms.

**Notas:**
- Los datos del perfil de usuario NO se copian a tablas locales. Solo se cachean en Redis durante la sesión activa.
- El TTL de 15 minutos en Redis garantiza que los cambios de perfil en el sistema corporativo se reflejan dentro de un tiempo razonable sin requerir invalidación manual.
